#### Shitpost Creator

Created by Mestik78
https://github.com/Mestik78


### Installation
1. Install python

2. Install pip

3. Install PIL
```
pip install image
```


### How to use:

1. Place the images you want to use on ``rawimages``
2. Place the words you want to use on ``words.txt``
3. run ``topng.py``
4. run ``shitpostCreator.py``
